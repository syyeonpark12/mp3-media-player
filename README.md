# C# WinForm 음악 플레이어

C# Windows Forms를 이용해 만든 음악 재생 프로그램입니다.

음악 재생과 일시정지, 이전 곡과 다음 곡 이동, 볼륨 조절, 재생목록 확인 기능을 구현했습니다.  
재생 중인 음악의 소리를 분석해 주파수에 따라 그래프가 움직이도록 만들었습니다.

## 프로젝트 소개

사용자가 재생목록에서 음악을 선택하면 앨범 이미지와 곡 정보가 화면에 표시되고 음악이 재생됩니다.

전체에는 음악 차트, 뉴스, 관리자 기능 등이 포함되어 있습니다.

## 메인 화면

음악 재생 화면과 재생목록을 한 화면에서 확인할 수 있도록 구성했습니다.

![메인 화면](./메인%20화면.png)

## 음악 재생 화면

앨범 이미지, 노래 제목, 재생 바, 볼륨 조절 버튼을 배치했습니다.

- 음악 재생 및 일시정지
- 이전 곡 및 다음 곡 이동
- 반복 재생
- 재생 위치 변경
- 볼륨 조절
- 음악에 맞춰 움직이는 그래프
- 유튜브 링크 이동

![재생 화면](./재생%20화면.png)

## 음악 재생목록

음악 제목, 가수, 앨범 이미지와 재생 시간을 목록으로 표시했습니다.

![음악 재생 목록](./음악%20재생%20목록.png)

## 실시간 음악 그래프

NAudio를 이용해 재생 중인 음악 데이터를 받아오고, FFT를 이용해 주파수 값으로 바꿨습니다.

주파수 값의 크기에 따라 막대그래프 높이가 달라지도록 구현했습니다.

![음악 주파수](./음악%20주파수.png)

![음악 주파수 구현 코드](./음악%20주파수%20구현%20코드.png)

## 유튜브 링크 기능

버튼을 누르면 설정된 유튜브 링크가 기본 브라우저에서 열리도록 구현했습니다.

![유튜브 버튼 링크](./유튜브%20버튼%20링크.png)

## 주요 기능

| 기능 | 설명 |

| 음악 재생 | 선택한 음악 파일 재생 |
| 재생 및 일시정지 | 버튼으로 음악 상태 변경 |
| 이전 곡 및 다음 곡 | 재생목록에서 곡 이동 |
| 볼륨 조절 | 슬라이더를 이용한 음량 변경 |
| 재생 위치 변경 | 재생 바를 이용한 위치 이동 |
| 재생목록 | 곡 제목, 가수, 앨범 이미지 표시 |
| 오디오 그래프 | 음악 주파수에 따라 그래프 변화 |
| 유튜브 링크 | 버튼 클릭 시 브라우저 실행 |
| 화면 디자인 | 버튼, 탭, 테두리와 배경 구성 |
